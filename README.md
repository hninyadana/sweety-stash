Sweety Stash is a personal finance application that makes budgeting fun by turning it into a game! It features a virtual pet mascot, "Sweety," whose mood changes based on your daily financial habits.

## Key Features

Data Persistence: unlike basic session-based apps, Sweety Stash saves your data permanently to a local JSON file (sweety_stash_data.json). Your budget, transactions, and streaks are safe even if you close the browser!

## Gamification:

- Mascot Moods: The cat reacts to your spending (Happy, Sad, or Excited).

- Streak Tracking: Build a streak by staying under budget daily.

- Rewards: Unlock visual badges (Spa Day, Vacation) for hitting milestones.

## Financial Tools:

- Monthly Setup: Define income and saving goals.

- Daily Logging: Track expenses with categories and descriptions.

- Fixed Expenses: Manage recurring bills like rent.

- Visual Reports: Interactive charts showing spending trends.
